# Public Assets Directory

## Required Images

Place your project images in this directory structure:

### Favicon
- `favicon.ico` - Your site favicon (in root of public/)

### Profile
- `avatar.jpg` - Your profile picture (400x400px recommended)

### Project Screenshots
Place these in the `projects/` subdirectory:

1. `projects/fixmyarea.jpg` - FixMyArea project screenshot (800x600px recommended)
2. `projects/agroinsight.jpg` - AgroInsight project screenshot (800x600px recommended)
3. `projects/opendesk.jpg` - OpenDesk project screenshot (800x600px recommended)
4. `projects/aplyo.jpg` - AplyO project screenshot (800x600px recommended)
5. `projects/gym.jpg` - Gym Website project screenshot (800x600px recommended)

## Image Guidelines

- Use JPG format for photos/screenshots
- Use PNG format for logos or images with transparency
- Optimize images before uploading (use tools like TinyPNG or ImageOptim)
- Recommended dimensions:
  - Avatar: 400x400px (square)
  - Project screenshots: 800x600px (4:3 aspect ratio)
  - Favicon: 32x32px or 16x16px

## Fallback Behavior

If images are not found, the application will fall back to placeholder images from picsum.photos.

## Next Steps

1. Add your `favicon.ico` to `public/`
2. Add your `avatar.jpg` to `public/`
3. Add project screenshots to `public/projects/`
4. Restart the development server to see your images
